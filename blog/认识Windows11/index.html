认识Windows11

发表于2023-11-23|更新于2023-12-07|Windows
|字数总计:603|阅读时长:1分钟

简单易用

探索新的 Windows 11 功能，帮助您以快速、有趣、轻松的方式完成工作。

通用搜索
选择任务栏上的搜索，然后开始输入您想要查找的内容。您将从您的文件、照片、联系人、电子邮件内甚至网络上获得个性化的搜索结果。

辅助功能
使用语音识别进行导航，通过实时字幕查看正在说的内容，或者让讲述人以自然的人声朗读屏幕上的文本。Windows 11 让每个人都可以轻松使用自己的 PC。

易于设置与个性化
个性化您的桌面主题、背景、颜色、文本等，以构建完美的创意空间来拓展您的兴趣爱好。

更高的工作效率

使用简单易用的工具和 Windows 11 的各项功能为您的桌面带来平衡，帮助您优化屏幕空间并最大限度地提高工作效率。

贴靠助手
快速整理屏幕上的内容，以便将您的最佳想法凸显出来。只需将窗口拖动到屏幕边缘，即可激活“贴靠助手”功能，将它们“贴靠”到清爽、有条理的网格中。

虚拟桌面组
使用虚拟桌面组，让您的数字生活井井有条，以适应您所有不同的工作和娱乐方式。自定义背景和标签可帮助您为生活的每个领域建立专用的桌面空间。

Microsoft 365
创新功能、高级应用程序、额外的云存储和高级安全性 — Windows 11 和 Microsoft 365 协同运作，帮助您发挥出最佳一面。

全面的安全保障

Windows 11 的功能让您能够创建、协作和保护您的资料。

兼顾性能与安全
自您开机那一刻起，Windows 11 已处于重重保护之下。它与您的硬件结合使用，且设计具有多个保护层，可以保护您的应用程序、信息和隐私安全。5

使用 SmartApp Control 保护设备
Windows 11 仅运行声誉良好且不会损害您的数据的应用程序。6

Microsoft Edge 安全性
Microsoft Edge 不仅可助您控制数据，防御恶意软件，还可扫描 Web，检查您的密码是否已被泄露。
